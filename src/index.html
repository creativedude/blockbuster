Blockbuster

Destroy the blocks by shooting them with a cannon. The more blocks you destroy, the higher your score will be. The game ends when you run out of blocks or when you run out of ammo.

Controls:

  • Use the slider to set the angle of the cannon.
  • Use the slider to set the windage of the cannon.
  • Hold the FIRE button to build power to shoot the cannon.
  • Press the RELOAD button to reload the cannon.
  • Use the direction keys to set windage and angle
  • Hold spacebar to build power and fire
  • Tap spacebar to reload the cannon

Good luck!
